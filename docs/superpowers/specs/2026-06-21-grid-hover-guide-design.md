# 打卡网格悬停定位设计

日期：2026-06-21

## 目标

在每日视图中增加轻量十字定位，帮助用户在长月份和多项目中确认当前格所在的项目行与日期列。

## 行为

- 仅每日视图启用。
- 鼠标进入可见打卡格时，记录当前项目 ID 和日期键。
- 当前项目行显示极浅灰底。
- 当前日期列显示极浅灰底。
- 当前格增加清晰描边。
- 左侧项目名称与顶部日期文字略微加深。
- 已打卡格保持黑色，只增加描边，不被浅灰覆盖。
- 鼠标离开网格后立即清除定位。
- 触摸设备不增加悬停效果。

## 实现边界

- 悬停状态只存在于 `ProjectGrid` 组件内，不写入本地数据。
- 使用行、列和当前格 CSS 类控制表现。
- 周/月汇总视图不响应悬停定位。
- 不改变点击、禁用未来日期和键盘可访问行为。

## 验收

- 悬停任意每日格时，对应日期表头和项目名称同步加深。
- 同行其他格和同列其他格显示极浅灰色提示。
- 鼠标移开后所有临时状态消失。
- 已打卡格仍保持黑色。
