# 字间排版快捷入口设计

## 目标

在页面右上角增加一个进入“字间排版”网页的快捷入口。

## 界面

- 按钮文字为“字”。
- 沿用现有 `icon-button shortcut-character` 样式以及日夜模式配色。
- 入口位于“盘”快捷入口之后、GitHub 图标之前。

## 行为

- 点击后在新标签页打开 `https://sange1022.github.io/zijian-text-layout/`。
- 使用 `noopener noreferrer` 保持外链安全。
- 无障碍名称和悬停提示均为“字间排版”。

## 验证

- 组件测试确认地址、新标签页和安全属性。
- 完整测试与生产构建通过。
- 浏览器确认线上按钮位置、文字与链接。
