# GoGoScrum 快捷入口设计

## 目标

在页面右上角的快捷入口区域增加 GoGoScrum 链接，让用户能从项目进度页面快速进入 `https://gogoscrum.com`。

## 界面

- 按钮显示单字“项”。
- 复用“巴”“芒”“饮”现有的字符快捷按钮样式。
- 放置在“饮”之后、日夜模式切换按钮之前。
- 浅色与深色模式下均沿用现有按钮颜色规则。

## 交互

- 点击后在新标签页打开 `https://gogoscrum.com`。
- 悬停提示及无障碍名称使用“GoGoScrum”。
- 链接使用 `noopener noreferrer`，避免新页面访问来源窗口。

## 验证

- 自动化测试确认按钮存在、网址正确，并使用新标签页安全属性。
- 运行完整测试与生产构建。
- 推送到 GitHub `main` 分支，并确认 GitHub Pages 部署成功。
