# GitHub 主页快捷入口设计

## 目标

在页面右上角增加一个可快速访问用户公开 GitHub 主页的入口。

## 界面

- 使用 `lucide-react` 的 GitHub 图标，不增加文字标签。
- 入口放在“盘”快捷入口之后、日夜模式按钮之前。
- 沿用现有 `icon-button` 尺寸、边框、悬停和深色模式样式。

## 行为

- 点击后在新标签页打开 `https://github.com/sange1022`。
- 链接使用 `noopener noreferrer`，避免新页面访问来源窗口。
- 无障碍名称和悬停提示均为“GitHub 主页”。

## 验证

- 组件测试确认链接地址、新标签页和安全属性。
- 完整测试与生产构建通过。
- 浏览器确认图标位置、页面渲染和点击跳转地址。
